Via [Jackson](<http://jaxn.org/blog/archives/794-XML-Formats-for-Office.html>), I see Microsoft has some great news:

> This actually is pretty big news. [Microsoft Office file formats are moving to XML](<http://www.microsoft.com/office/xml/default.mspx>). These are formats that all developers can use. This means it will be much easier for web applications to create and process MS Office formats. This also means that it will be easy for open source projects to offer compatibility with MS Office.
