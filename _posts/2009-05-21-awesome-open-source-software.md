In preparation for my presentation ([Try some Open Source.... Everybody's doing it](<http://blog.centresource.com/2009/05/21/try-some-open-source-everybodys-doing-it/>)), I've compiled some of the best offerings in the land of OSS.

**Utilities**  
Firefox ([http://www.mozilla.com](<http://www.mozilla.com/>)) - Browser  
Azureus ([http://azureus.sourceforge.net](<http://azureus.sourceforge.net/>)) - BitTorrent  
FileZilla ([http://www.FileZilla.com](<http://www.filezilla.com/>)) - FTP Client  
CLAMWIN ([http://www.clamwin.com](<http://www.clamwin.com/>)) - Open Source Anti-Virus  
TightVNC ([http://www.tightvnc.com](<http://www.tightvnc.com/>)) - Remote computer control

**Communication**  
Pidgin ([http://www.pidgin.im](<http://www.pidgin.im/>)) - Multi Chat  
Thunderbird ([http://www.mozillamessaging.com](<http://www.mozillamessaging.com/>)) - Email Suite  
Wordpress ([http://www.wordpress.org](<http://www.wordpress.org/>)) - Blog Platform

**Entertainment**  
Miro ([http://www.getmiro.com](<http://www.getmiro.com/>)) - Media Player (HD Content)  
Tremulous ([http://tremulous.net](<http://tremulous.net/>)) - First Person Shooter Game  
LinCity (<http://lincity-ng.berlios.de/wiki/index.php/Main_Page>) - Sim City  
FlightGear ([http://flightgear.org](<http://flightgear.org/>)) - Flight Simulator

**Graphics**  
Paint.net ([http://www.getpaint.net](<http://www.getpaint.net/>)) - Image Editing  
GIMP ([http://www.gimp.org](<http://www.gimp.org/>)) - Image Editing  
Blender 3D ([http://www.blender.org](<http://www.blender.org/>)) - 3D Simulator  
Inkscape ([http://www.inkscape.org](<http://www.inkscape.org/>)) - Vector Graphics Editor (like Illustrator/CorelDraw)

**Audio/Video**  
MediaCoder (<http://mediacoder.sourceforge.net/dl/dvdcss.html>) - Media Transcoder  
Handbrake ([http://handbrake.fr](<http://handbrake.fr/>)) - DVD Backup  
Audacity ([http://audacity.sourceforge.net](<http://audacity.sourceforge.net/>)) - Multi Track Audio Editor  
VLC Media Player (<http://www.videolan.org/vlc/>) - Super Media Player, Caster

**Productivity**  
OpenOffice ([http://www.openoffice.org](<http://www.openoffice.org/>)) - Office Suite  
Bluemarine ([http://bluemarine.tidalwave.it](<http://bluemarine.tidalwave.it/>)) - Photo Manager

**Enterprise**  
SugarCRM ([http://www.sugarcrm.com](<http://www.sugarcrm.com/>)) - Enterprise CRM  
Drupal ([http://www.drupal.org](<http://www.drupal.org/>)) - Web Publishing, Content Management  
Asterisk ([http://www.asterisk.org](<http://www.asterisk.org/>)) - Open Source Phone System  
Zarafa ([http://zarafa.com](<http://zarafa.com/>)) - Exchange Competitor
