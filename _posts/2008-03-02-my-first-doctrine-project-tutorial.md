Over at the [Doctrine PHP ORM](<http://www.phpdoctrine.org/>) website they have just released a tutorial on how to get started using Doctrine in under 5 minutes!!!

> "I have put together a short and sweet [tutorial](<http://trac.phpdoctrine.org/wiki/MyFirstProject>) on how to get started using Doctrine in under 5 minutes. A few weeks ago, we announced the [sandbox package](<http://www.phpdoctrine.org/download/0_10/2/format/tgz/sandbox/1>) which allows you to get started using Doctrine with a zero configuration sandbox environment. The tutorial is built using this sandbox package and shows you how Doctrine can be easily implemented and used in your web applications."

The tutorial can be found [here](<http://trac.phpdoctrine.org/wiki/MyFirstProject>)
