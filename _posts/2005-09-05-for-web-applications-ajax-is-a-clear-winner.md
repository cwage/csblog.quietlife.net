CentreSource's development team focuses exclusively on Web-based Software. As such, we take great pleasure in seeing the popularity of [AJAX ](<https://web.archive.org/web/20050903235519/http://en.wikipedia.org/wiki/AJAX>)grow. In the past, web applications have been seen as having boring and limited user interfaces. AJAX, which is actually an old technology, is changing the face of web applications due to new and innovative ways of using it.

Wired magazine has [a great article](<https://web.archive.org/web/20050814002701/http://www.wired.com/news/technology/0,1282,68403,00.html>) about AJAX's climbing popularity & the ways it may affect the future of software...

> Experts say they don't expect AJAX will make desktop applications obsolete. But it has already affirmed the viability of the web as a standalone software development platform.
> 
> "This is going to go a long way towards eliminating the user interface insults and injuries we have suffered since we moved to the web," O'Reilly�s Dornfest said. "Now people these days expect it to be flat so they might be a little surprised (by AJAX applications). But the rest of us see AJAX and say 'Ahh, this is what it is supposed to be like.'"
