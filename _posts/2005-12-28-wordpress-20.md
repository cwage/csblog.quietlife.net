[Wordpress 2.0](<https://web.archive.org/web/20051228134015/http://wordpress.org/download/>) has been released. I've gone ahead and upgraded the Centresource Blog and all appears well. The only problem I've noticed was with the [feedburner redirection](<https://web.archive.org/web/20051229024028/http://orderedlist.com/articles/wordpress-feedburner-plugin/>) plugin we use to redirect RSS requests to [Feedburner](<https://web.archive.org/web/20070602050109/http://www.feedburner.com/>). I've fixed this with some manual .htaccess redirections for now.

There's no official Changelog (yet), but you can see a list of some of the changes [here](<https://web.archive.org/web/20060406140727/http://asymptomatic.net/2005/11/29/2135/whats-new-in-wordpress-20>). A lot of the changes were on the backend, which is why I am pleased to see that a majority of the plugins we're using are still operational. This includes [WordVerify](<https://web.archive.org/web/20060118042052/http://chris.quietlife.net/WordVerify>), which I've added to [the list](<http://codex.wordpress.org/User:Matt/2.0_Plugin_Compatibility>) of plugins known to work with Wordpress 2.0.

Incidentally, since I have installed WordVerify on both our blogs, I haven't received a single comment-spam -- moderated or otherwise. I've even disabled the other blacklist plugins as they seem largely unnecessary. Who knows how long this will last, but it's nice for the time being.
