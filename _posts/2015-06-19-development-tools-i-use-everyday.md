When it comes to development, I am a simple man who enjoys using the bare minimum when when it comes to apps. I personally think that when it comes to software on my development machine, I want as few extra things as possible. I do this for a few reasons. Firstly, I don't want a lot of apps because I am very conscious of my laptops memory and disk space. Secondly I personally don't like to have apps that duplicate functionality, so often times I pick apps that either do one thing really well, or can give me a lot of functionality with little trouble. Lastly, I like to have my dev environment as close to as my production machine as possible.

[Alfred](<http://www.alfredapp.com/>) - [![Alfred](https://csblog.quietlife.net/wp-content/uploads/2015/06/alfred-150x150.png)](<http://www.alfredapp.com/>)Alfred is a tool that revolutionized my productivity the moment I downloaded it. It is free forever with limited functionality. That personally I loved, and used for months. It was a better spotlight app, then the mac default. I used it for quick searches, switching between apps, and general increase of speed when getting between the many process that I am doing. Then a coworker showed me the paid version, which I was hesitant for the reason that I loved the free version, and was sceptical about how much better it could get, which I was gladly wrong. With a library of helpful workflows that made my already quick workflow even faster, and the ability to create my own, it makes it hard if not impossible for alfred to be one of the first apps I download. Plus its connection to dropbox means that I don't have to adjust my settings every time.

[Size Up](<http://www.irradiatedsoftware.com/sizeup/>) - [![SizeUp ](https://csblog.quietlife.net/wp-content/uploads/2015/06/app-sizeup@2x-150x150.png)](<http://www.irradiatedsoftware.com/sizeup/>)Size Up is a window management tool that allows you set and use your own predefined window spaces. I often have many things open between my text editor, google chrome, slack, and any number of other things that are pertinent to my current task. Size Up allows me to quickly change the size of my windows, and make it so that i can quickly see everything I'm working on or with.

[Screenhero](<https://screenhero.com/>) - [![Screenhero](https://csblog.quietlife.net/wp-content/uploads/2015/06/screenhero-logo-150x150.png)](<https://screenhero.com/>)Screenhero is a great tool for pair programming. Screenhero allows me to control or give control to a coworker, where we can work together on a project, help each other with bugs, or any other problem. The great thing about this is that instead of focusing on the face of the person you are working with, it focuses on the code and the voice of the person you are working with. The simple way that it allows you to work with someone makes it a great app in my opinion.

[dash](<http://kapeli.com/dash>) - [![Dash](https://csblog.quietlife.net/wp-content/uploads/2015/06/dash-150-150x150.png)](<http://kapeli.com/dash>)dash is one of the best resources for programming documentation. With Dash I can download lots of documentation, for all the languages that I use, as well as many of the tools. This is great so that I can see the offical documentation without having to go online and even cross check it with other languages' method. This improves my overall productivity when I'm writing code, because instead of going online and googling and then going down all the rabbit holes the internet bring, I get the information I need and get back to work.
