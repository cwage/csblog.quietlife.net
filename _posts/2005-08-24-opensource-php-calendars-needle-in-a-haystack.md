Over the past 2 years, we've had multiple clients inquire about having a simple calendar on their website. After implementing a few solutions from various vendors, I realized that there aren't many good open-source calendars in existence.

What I did notice, however, is that there is a wide variety of vendors selling calendar systems as stand-alone applications or encoded code to download & install on your web server.

By far, my favorite calendar to date is http://www.easyphpcalendar.com - but its encoded and doesn't allow integration with custom software. For a basic website, I think this solution is WELL worth the $20 ($50 if you want their name to disappear from all instances).

As for the 'free' calendars, I've found the following (note, I did not test these):

<https://web.archive.org/web/20050824/http://www.phpcalendars.com/ (phpCalendar)>  
* Not sure if it has an admin panel
